# 赚钱效应 · 形态规则（规则版 / v0）

目标：先可用、可解释，后续再用样本回看微调阈值。

## 数据
- 使用近 20 个交易日窗口（PRD）
- 形态识别优先基于 Top200 个股做（降低 API 调用）

## 规则（初版）

### 1) 连板
- 定义：连续 N 天涨停
- 近似实现（不依赖涨停价字段）：
  - 主板：`pct_chg >= 9.8`
  - 创业/科创：`pct_chg >= 19.8`
- 连板数：从今天往前数连续满足条件的天数

### 2) 首板
- 今日为涨停，且昨日不为涨停（连板数=1）

### 3) 120 日新高/历史新高（待 v1）
- 需要更长窗口（120/250）与更稳定的数据缓存；v0 先不做。

### 4) 低位突破（v0）
- 条件：
  - 今日收盘价 > 过去 20 日最高收盘（不含今日）
  - 且今日收盘价位于过去 120 日区间的下 30%（需要 120 日窗口；v0 先简化为 20 日区间下 30%）

### 5) 低位首板（组合标签）
- 首板 且 低位突破

备注：v0 先把“连板/首板”做出来并上线；其余形态逐步补齐。
